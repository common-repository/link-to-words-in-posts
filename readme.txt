This is basically a plugin that shorten the hassel of adding links of words occuring multiple times in a post from simple panel.

== Description ==
 
This Plugin will help wordpress posts editors to add links to any keywords in the selected post from a simple 
panel.This will help the bloggers to linking the keywords from simple panel with all attributes of a link. 
 <ul>
 <li> This plugin will only add links to keywords with few attributes.</li>
 <li> This plugin will add links to keywords as many time as keyword occurs in a selected post.</li> 
 <li> Title of Link can be added. </li>
 <li> Relative Attribute can be set as per need.</li>
 <li>Target of the link can be set using the same panel. </li>
 </ul>

Updates for more links attributes are comming soon.

== Installation ==

This section describes how to install the plugin and get it working.
<ol>
<li>Just login to the wordpress admin.</li>
<li>Goto plugins / Upload the Plugin folder to wp-content/plguins folder</li>
<li>Install new Plugin</li>
<li>Upload a plugin </li>
<li>Activate.</li>
</ol>

You will see a new tab 'Keyword Linking' under General Settings.

== Frequently Asked Questions ==

Q. Will this plugin helps me updating the selected post?<br>
A. Yes This Plugin will only allow you to add links to the keywords of selected posts from a drop down.
<br>
Q. Will this plugin also allow to add links for page content ?<br>
A  No, Current version of the plugin is only for the default wordpress posts.
